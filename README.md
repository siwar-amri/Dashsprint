# DashSprint: Jira KPI Dashboard

**DashSprint** is a dynamic dashboard application designed for Jira teams to monitor and track their KPIs over time. It provides real-time insights from Jira accounts, helping teams make data-driven decisions and improve project performance.

## Features

* Connects dynamically to Jira accounts
* Tracks and visualizes team KPIs over time
* Interactive charts and tables for performance metrics
* User-friendly dashboard for quick insights

## Technology Stack

* **Backend:** Spring Boot
* **Database:** PostgreSQL
* **Frontend:** Angular

## Installation & Setup

1. **Clone the repository:**

   ```bash
   git clone https://github.com/YOUR_USERNAME/DashSprint.git
   cd DashSprint
   ```

2. **Backend Setup (Spring Boot):**

   * Configure `application.properties` with your Jira API credentials and PostgreSQL database details.
   * Run the Spring Boot application:

     ```bash
     ./mvnw spring-boot:run
     ```

3. **Frontend Setup (Angular):**

   * Navigate to the `frontend` directory:

     ```bash
     cd frontend
     ```
   * Install dependencies:

     ```bash
     npm install
     ```
   * Run the Angular development server:

     ```bash
     ng serve
     ```
   * Access the dashboard at `http://localhost:4200`

## Usage

* Connect your Jira account through the provided interface.
* Select the projects or teams you want to monitor.
* View dynamic KPIs over time using charts and tables.
